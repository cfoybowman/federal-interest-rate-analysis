This project explores U.S. monetary policy behavior using historical federal funds rate data.  
The analysis focuses on rate changes, policy cycles, and relationships with unemployment and inflation.

### Key Features
- Time-series analysis of federal funds rate levels and changes
- Rolling averages to identify tightening and easing cycles
- Automatic classification of policy regimes
- Comparison of rate changes to unemployment and inflation

### Tools Used
- Python (pandas, matplotlib)
- Jupyter Notebook
- Kaggle economic data

### Key Insight
Federal Reserve policy is largely counter-cyclical, with easing occurring more frequently than tightening and aggressive interventions clustering around periods of economic stress.
